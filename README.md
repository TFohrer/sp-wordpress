# X-Child theme for sp website


## Getting started 

Copy child theme into default wordpress themes folder (`wp-content/themes`)

<br>

Install dependencies
  ```
    yarn install 
``` 
<br>
Build

```
    yarn build 
    # or
    yarn watch 
``` 
<br>
Build for prod

```
    yarn dist
```


### Features

- Babel transpiler 
- PostCSS autoprefixer 
- UglifyJS
